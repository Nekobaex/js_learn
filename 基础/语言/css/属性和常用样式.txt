继承属性
  inherit 继承
  initial 不默认

border:
  边框:
    border: 1px solid red

    border-width
    border-style: solid / dotted / dashed
    border-color

  圆角:
    border-radius: 1px;
    border-radius: 1px 2px;    // 左对角, 右对角
    border-radius: 1px 2px 3px 4px;  // 左上, 右上, 右下, 左下 (从左上开始顺时针)

    border-top-left-radius: 1px
    border-top-right-radius: 2px 
    border-bottom-right-radius: 3px    
    border-bottom-left-radius: 4px

边框外阴影:
  (圆角和 border 同步)
  box-shadow: 5px 9px 3px 20px #000;  // x轴, y轴, 模糊半径, 阴影半径

  可以用逗号分隔多个阴影:
    box-shadow: 5px 5px 20px 5px darkslateblue, -5px -5px 20px 5px dodgerblue,
    inset 0px 0px 10px 2px darkslategray, inset 0px 0px 20px 10px steelblue;

文字外阴影:
  text-shadow: 5px 9px 20px #000; // 和 box-shadow 相比, 没有模糊半径

  也可以和 box-shadow 一样, 用逗号分隔多个阴影
