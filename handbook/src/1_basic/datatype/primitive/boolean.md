# `Boolean`
仅有两个值 `true` 和 `false`,  一般用于流程控制之中

比较运算:
  ```js
    // 数字之间的比较, 可以产生 true 或 false 的结果

    // ==  和 !=  表示相等比较, 比较过程中会进行隐式转换
    123  == 123n
    0    == false
    // === 和 !== 表示严格相等比较, 比较过程中不会发生隐式转换
    123 !== 123n
    0   !== false

    // 大小比较: >, >=, <, <=
  ```

逻辑运算:
  ```js
  // true 和 false 之间的运算

  // || 和 && 实际上是短路运算
  // 左边的表达式能完全确定结果, 则不会访问右边的表达式

  // 或 ||
  // 与 &&
  // 非 !

  ```
